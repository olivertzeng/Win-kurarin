# Win-kurarin
## Windows x きゅうくらりん
Win-kurarin tries to create a lot of windows to recreate the animation in きゅうくらりん on
Windows.
Watch this video to learn more: www.nicovideo.jp/watch/sm41820938

## Linux Support(with Wine/Proton)
Currently, Win-kurarin is only designed to be only used on Windows, but it could partially work on
Linux with Wine or Proton too! Although the program crashes halfway through.
